# Juego matemático con formatos garantizados

Cambios:
- Filas y columnas reorganizadas para que todo quede unido.
- Cada línea es una operación válida: A operador B = Resultado.
- Solo usa valores enteros de 1 a 99.
- No usa división para evitar casos incorrectos.
- Tiene selector de formato:
  1. Cruz conectada
  2. Escalera
  3. Doble bloque unido
- No sobran números: la cantidad de números disponibles coincide con las casillas vacías.
- Arrastre compatible con celular. Si se suelta fuera de una casilla, vuelve a la bandeja.

Subir a Hetzner:
Subir index.html, style.css y app.js a la carpeta pública del sitio.
