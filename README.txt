# Juego matemático corregido

Correcciones:
- Crucigrama automático en una sola estructura conectada.
- No hay filas/columnas sueltas.
- No genera divisiones ni cálculos inválidos.
- No sobran números: hay la misma cantidad de números que casillas vacías.
- Al arrastrar, el número no queda flotando: si se suelta fuera de una casilla vuelve a la bandeja.
- Compatible con celular y PC.

Subir index.html, style.css, app.js y levels.js a la carpeta pública del sitio.
